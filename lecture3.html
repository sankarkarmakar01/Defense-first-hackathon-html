What is List in HTML? üìù

In HTML, a list is a way to group related items together. Lists are commonly used for navigation menus, instructions, features, or any set of items that belong together.

List Elements (<ul>, <ol>, <li>)

These tags are used to create lists. This is a perfect example of nesting.

  • <ul>: An Unordered List. It creates a bulleted list.

  • <ol>: An Ordered List. It creates a numbered list.

  • <li>: A List Item. Each item in either type of list must be wrapped in an <li> tag.

The Purpose: To group related items together in a list format.

The <li> elements are nested as children inside the <ul> or <ol> parent.

Unordered List ( <ul> - Bulleted )

  • Tea Bag
  • Water
  • Sugar
  • Milk

Exmaple:

  <ul>
  <li>Tea Bag</li>
  <li>Water</li>
  <li>Sugar</li>
  <li>Milk</li>
  </ul>

Ordered List ( <ol> - Numbered )

  1. First, boil the water in a kettle.
  2. Add tea bag, sugar and milk into it
  3. Keep boiling it for 5 minutes
  4. Serve the tea by pouring it into cup

Exmaple:

  <ol>
  <li>First, boil the water in a kettle.</li>
  <li>Add tea bag, sugar and milk into it</li>
  <li>Keep boiling it for 5 minutes </li>
  <li>Serve the tea by pouring it into cup</li>
  </ol>

Nested List ( List inside another List )

A nested list means placing one list (<ul> or <ol>) inside another list item. This is useful for showing sub-items, categories, or steps with details.

  • Fruits
    • Apple
    • Mango
    • Banana
  • Vegetables
    1. Carrot
    2. Potato
    3. Tomato

Exmaple:

  <ul>
    <li>Fruits
      <ul>
        <li>Apple</li>
        <li>Mango</li>
        <li>Banana</li>
      </ul>
    </li>
    <li>Vegetables
      <ol>
        <li>Carrot</li>
        <li>Potato</li>
        <li>Tomato</li>
      </ol>
    </li>
  </ul>

What is Table in HTML? üìä

In HTML, a table is used to display data in a structured row and column format. Tables are commonly used for timetables, product details, comparison charts, invoices, or any data that fits into a grid.

Table Elements (<table>, <tr>, <td>, <th>, <thead>, <tbody>, <tfoot>)

These tags are used to build a table structure. Each serves a specific role.

  • <table>: Defines the table itself. It acts as the parent container.

  • <tr>: A Table Row. Used to create a new row inside the table.

  • <td>: A Table Data cell. It holds the actual data inside a row.

  • <th>: A Table Header cell. It defines a heading for a column, usually displayed in bold.

  • <thead>: Groups the header content of the table. Usually contains one or more rows of <th>.

  • <tbody>: Groups the main body (data) content of the table.

  • <tfoot>: Groups the footer content of the table. Typically used for totals or summary rows.

The Purpose: To arrange and present data in a tabular (row & column) structure with clear separation of header, body, and footer.

The <thead>, <tbody>, and <tfoot> elements are nested inside the <table>, making the table more organized and semantic.

Basic Table Example with thead, tbody, and tfoot

Subject Marks
Math 95
Science 90
English 88
Total 273

Example:

  <table border="1" cellpadding="8" cellspacing="0">
    <thead>
      <tr>
        <th>Subject</th>
        <th>Marks</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>Math</td>
        <td>95</td>
      </tr>
      <tr>
        <td>Science</td>
        <td>90</td>
      </tr>
      <tr>
        <td>English</td>
        <td>88</td>
      </tr>
    </tbody>
    <tfoot>
      <tr>
        <th>Total</th>
        <th>273</th>
      </tr>
    </tfoot>
  </table>
